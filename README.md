# Vision AI Tracker

Aplicação de visão computacional desenvolvida em Python capaz de identificar e rastrear gestos em tempo real utilizando webcam.

Projeto desenvolvido durante o NLW da Rocketseat, aplicando conceitos de inteligência artificial, processamento de imagem e execução em tempo real.

---

## 🚀 Funcionalidades

- Captura de vídeo em tempo real via webcam
- Reconhecimento de gestos utilizando visão computacional
- Processamento de imagem em tempo real
- Interface para visualização dos resultados

---

## 🧠 Tecnologias utilizadas

- Python
- OpenCV
- Machine Learning
- Computer Vision
